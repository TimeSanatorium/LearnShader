用代码创建文件的命令行是				echo"内容">文件名.文件后缀名
输出某文件内的所有内命令行是				cat 文件名字.文件后缀名
提交多个文件夹上去	 				git add 文件名.后缀 文件名.后缀	
版本回退命令					git reset --soft 版本ID
比较版本差异默认比较的是工作区和暂存区的差异		git diff						这里还能加上文件名用来比较具体文件的差异
							HEAD		工作区和库之间的差异	
							--cached		暂存区和库之间的差异
							版本ID1 版本ID2 	比较两个版本之间的差异内容
							分支名1 分支名2 	比较两个分支之间的差异
